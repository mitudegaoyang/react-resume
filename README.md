# react-resume

基于 react-antd-admin 搭建的前端简历网站 [预览地址](https://me.gaotianyang.top)

## 功能

- [x] 首页
  - [ ] 首页 logo
  - [ ] three.js 背景
- [x] 工作经历
- [x] 项目经历
  - [ ] 图片放大
- [x] 擅长技术
- [x] 自我评价
- [x] 作者博客
- [x] 项目 header
  - [ ] 个人头像

## 涵盖知识点

- 原生 JS 实现钉钉官网滚动特效
- echarts-wordcloud 绘图
- echarts-wordcloud maskImage 正确使用
- axios、lodash、dayjs 使用
- CSS3 的 animation 动画
